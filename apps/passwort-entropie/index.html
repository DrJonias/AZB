How calculation works

The tool estimates password strength using these assumptions:

  • The more character sets used, the larger the character pool.
  • Entropy is calculated as password length × log2(pool size).
  • Entropy is used to derive the number of possible combinations.
  • The displayed times estimate how long an attacker would need at different speeds.

Note: this is a mathematical estimate, not a full security audit.
